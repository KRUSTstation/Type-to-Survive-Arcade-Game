# Type to Survive (NJC Funtasia)

A fast-paced Roblox typing game where players must type words as quickly and accurately as possible to stay alive and achieve the highest score.

Inspired by the typing game featured in **Cyberthon**, Type to Survive challenges players to improve their typing speed, accuracy, and reaction time in an engaging and competitive environment.

## Features

- Fast-paced typing gameplay
- Accuracy and speed-based scoring
- Competitive high-score chasing
- Increasing difficulty as the game progresses
- Simple and accessible gameplay for all players

## Gameplay

1. Join the game.
2. A word appears on the screen.
3. Type the word correctly as quickly as possible.
4. Earn points for correct entries.
5. Survive for as long as you can and aim for the highest score.

## Inspiration

This project was inspired by the typing game played during **Cyberthon**. The goal was to recreate the excitement of rapid typing challenges while adding a Roblox-based experience. The main purpose of this game was to raise funds for Eden Friendship Club during NJC Funtasia 2026.

## Technologies Used

- Luau (Roblox Lua)
